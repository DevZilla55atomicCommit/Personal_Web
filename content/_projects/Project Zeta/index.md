## Overview

Project Zeta is a productivity application featuring drag-and-drop kanban boards, team collaboration features, and real-time updates. Perfect for agile teams working on projects together.

## Key Features

- **Kanban Boards:** Create and manage boards with drag-and-drop cards
- **Team Collaboration:** Comment on cards, assign tasks, set due dates
- **Real-time Updates:** See changes as they happen across all devices
- **Mobile First:** Works great on any device, anywhere

## Technology Stack

- **Frontend:** React, TypeScript, Redux
- **Backend:** Node.js, Socket.io
- **Database:** MongoDB, Redis
